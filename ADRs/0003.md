# ADR 0003: Which algorithim to use for JWT

## Status

Accepted

## Context

It would be safer to use private and public key for authentication instead of 1 key (Use RS256).

## Decision

we will use 1 key because this repo is not going out to a public domain.
this code will also not be shared to the public, so the keys are safe.
secondly, when we do share it, then it would be on a centrilized server, not on multiple

## Consequences

Positive consequences: Easier implementation
Negative consequences: Authenticating is not very safe, because when the server gets hacked, all tokens are controlled by an untrusted source.
